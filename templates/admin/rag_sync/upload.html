What happens after upload?
Each file is saved to the server's company_docs/ folder with a unique name, a database record is created, and a sync task is queued to upload the file to OpenAI and attach it to the Vector Store. The sync status will update from Pending → Uploading → Indexed automatically.

Duplicate files (identical content) are detected by SHA-256 hash and skipped with a warning — no duplicate records are created.
